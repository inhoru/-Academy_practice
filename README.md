## 🖤 기능요구사항
기본적으로 학생정보를 등록해 관리하는 프로그램이다.

- 학생등록 올바르게 입력했을경우 등록성공 출력

   - 등록할 학생이 중복될경우 등록실패 출력
   - 등록할 공간이없으면 등록실패 출력

- 학생 조회

   - 등록된학생 정보 전체 출력

- 이름으로 조회

   - 등록된 학생 이름으로 조회시 전체정보 출력

- 학생 수정

    - 수정할 학생 학번 출력후
    - 수정할 학생 학번 입력
    - 학번 일치할시 학년, 전공, 주소 수정가능

- 학생 삭제기능

   - 등록된 학생정보 삭제
   - 삭제한 공간에 새로운 학생등록가능
